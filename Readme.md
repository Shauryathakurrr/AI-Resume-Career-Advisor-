# AI Resume & Career Advisor 🚀

## 📌 Overview

This project is a Generative AI-based application that analyzes resumes, provides improvement suggestions, identifies skill gaps, and generates interview questions.

## 🧠 Features

* Resume Analysis with Score 📊
* Resume Improvement ✨
* Skill Gap Analysis 🧠
* Interview Preparation 🎤

## ⚙️ Tech Stack

* Python
* Streamlit (GUI)
* Ollama (Llama3:instruct Model)
* Matplotlib

## ▶️ How to Run

### 1. Install dependencies

pip install -r requirements.txt

### 2. Run Ollama

ollama serve

### 3. Run App

python -m streamlit run app.py

## ⚠️ Note

This project uses a local AI model (Llama3:instruct via Ollama), so internet is not required after setup.
